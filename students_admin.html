Quick Reference

• Chest Number is the unique ID students use to search their profile.
• Auto Score is calculated from results — 5pts for 1st, 3pts for 2nd, 1pt for 3rd.
• Bonus Points are manually added on top of auto-calculated score.
• Total Score = Auto Score + Bonus Points.

After adding results in the Results Admin, student scores update automatically on the public page.

📥 CSV Import

Upload a CSV file with columns:
name, chest_number, class, team_id, extra_score

CSV Format Example
name,chest_number,class,team_id,extra_score
Ahmed Ali,101,10A,1,0
Fatima Zahra,102,9B,2,0
Mohammed Salim,103,10B,1,5
Aisha Noor,104,8A,3,0

• team_id: 1=Alpha, 2=Beta, 3=Gamma, 4=Delta
• extra_score: bonus points (use 0 if none)
• Duplicate chest numbers will be skipped
